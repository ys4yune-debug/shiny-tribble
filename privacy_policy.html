개인정보 처리방침

최종 수정일: 2025년 11월 4일

1. 개요

할일뭐였지? (Whattodo, 이하 "본 앱")는 사용자의 개인정보를 중요하게 생각하며, 개인정보 보호법 및 관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 본 앱이 수집하는 정보, 사용 목적, 보관 방법에 대해 설명합니다.

2. 수집하는 개인정보

2.1 할일 데이터

  • 수집 항목: 사용자가 입력한 할일 제목, 카테고리, 마감일, 반복 설정
  • 수집 목적: 할일 관리 서비스 제공
  • 저장 위치: 사용자 기기 내부 저장소 (SharedPreferences)
  • 외부 전송: 없음 (모든 데이터는 기기 내부에만 저장됨)

2.2 캘린더 데이터

  • 수집 항목: 기기의 캘린더 일정 정보 (제목, 날짜, 시간)
  • 수집 목적: 오늘과 내일의 일정을 잠금화면에 표시
  • 접근 방법: Android 캘린더 권한 (READ_CALENDAR)
  • 저장: 일시적으로 메모리에만 로드되며, 별도로 저장하지 않음
  • 외부 전송: 없음

2.3 앱 설정 정보

  • 수집 항목: 테마 설정, 폰트 크기, 언어 설정 등
  • 수집 목적: 사용자 맞춤 환경 제공
  • 저장 위치: 기기 내부 저장소 (SharedPreferences)
  • 외부 전송: 없음

2.4 광고 식별자

  • 수집 항목: Google 광고 ID (익명)
  • 수집 목적: 개인 맞춤형 광고 제공
  • 처리 주체: Google AdMob
  • 관련 정책: Google 개인정보처리방침

3. 개인정보의 사용

수집된 정보는 다음의 목적으로만 사용됩니다:

  • 할일 관리 기능 제공
  • 캘린더 일정 표시
  • 사용자 맞춤 설정 유지
  • 앱 내 광고 표시 (AdMob)
  • 서비스 개선 및 버그 수정

4. 개인정보의 보관 및 파기

4.1 보관 기간

  • 할일 데이터: 사용자가 삭제할 때까지 또는 앱 삭제 시까지
  • 앱 설정: 앱 삭제 시까지
  • 캘린더 데이터: 별도 저장하지 않음 (임시 로드만)

4.2 파기 방법

  • 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다
  • 설정에서 "설정 초기화" 기능으로 앱 설정 데이터를 초기화할 수 있습니다

5. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 외부에 전송하거나 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외입니다:

5.1 Google AdMob

  • 제공 목적: 광고 표시
  • 제공 항목: 광고 식별자 (익명)
  • 제공 받는 자: Google LLC
  • 관련 정책: Google AdMob 개인정보처리방침

6. 사용자 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 열람 권리: 앱 내에서 본인의 할일 데이터를 확인할 수 있습니다
  • 수정 권리: 할일 데이터를 자유롭게 수정할 수 있습니다
  • 삭제 권리: 할일 데이터를 삭제하거나 앱을 삭제할 수 있습니다
  • 처리 정지 권리: 앱 삭제 또는 권한 철회로 데이터 처리를 중단할 수 있습니다

7. 권한 사용 안내

7.1 필수 권한

  • 접근성 서비스 (BIND_ACCESSIBILITY_SERVICE)
    • 목적: 잠금화면 대체 기능 제공
    • 수집 데이터: 없음 (기능만 사용)

7.2 선택 권한

  • 캘린더 읽기 (READ_CALENDAR)
    • 목적: 오늘과 내일의 일정 표시
    • 거부 시: 캘린더 기능 사용 불가 (앱의 다른 기능은 정상 작동)

8. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.

9. 개인정보 보호책임자

책임자: 윤유식 (개인 개발자)

문의처: ys4yune@gmail.com

10. 개인정보처리방침 변경

본 개인정보처리방침은 법령 및 정책 변경, 서비스 개선에 따라 변경될 수 있습니다. 변경 시 앱 내 공지사항 또는 업데이트 노트를 통해 안내드립니다.

11. 데이터 보안

본 앱은 사용자 데이터 보호를 위해 다음의 보안 조치를 취하고 있습니다:

  • 모든 데이터는 기기 내부에만 저장
  • Android 샌드박스 보안 모델 적용
  • 외부 서버로 데이터 전송 없음
  • 정기적인 보안 업데이트

12. 데이터 이전

본 앱은 사용자의 개인정보를 해외로 이전하지 않습니다. (단, Google AdMob의 광고 ID는 Google의 정책에 따라 처리됩니다)

본 개인정보처리방침은 2025년 11월 4일부터 적용됩니다.
문의사항이 있으시면 ys4yune@gmail.com으로 연락 주시기 바랍니다.

Privacy Policy

Last Updated: November 4, 2025

1. Overview

Whattodo (hereinafter referred to as "the App") values your privacy and complies with personal information protection laws and related regulations. This Privacy Policy explains what information the App collects, its purpose of use, and how it is stored.

2. Information We Collect

2.1 To-Do Data

  • Collected Items: To-do titles, categories, due dates, and repeat settings entered by users
  • Purpose: Providing to-do management services
  • Storage Location: Device internal storage (SharedPreferences)
  • External Transfer: None (all data is stored only on the device)

2.2 Calendar Data

  • Collected Items: Calendar event information (title, date, time)
  • Purpose: Displaying today's and tomorrow's events on the lock screen
  • Access Method: Android Calendar permission (READ_CALENDAR)
  • Storage: Temporarily loaded in memory only, not stored separately
  • External Transfer: None

2.3 App Settings

  • Collected Items: Theme settings, font size, language settings, etc.
  • Purpose: Providing personalized user experience
  • Storage Location: Device internal storage (SharedPreferences)
  • External Transfer: None

2.4 Advertising Identifier

  • Collected Items: Google Advertising ID (anonymous)
  • Purpose: Providing personalized advertisements
  • Processor: Google AdMob
  • Related Policy: Google Privacy Policy

3. Use of Personal Information

Collected information is used only for the following purposes:

  • Providing to-do management features
  • Displaying calendar events
  • Maintaining user preferences
  • Displaying in-app advertisements (AdMob)
  • Service improvement and bug fixes

4. Retention and Disposal of Personal Information

4.1 Retention Period

  • To-Do Data: Until deleted by user or app uninstallation
  • App Settings: Until app uninstallation
  • Calendar Data: Not stored separately (temporary load only)

4.2 Disposal Method

  • All data is automatically deleted when the app is uninstalled
  • App settings data can be reset using the "Reset Settings" feature in Settings

5. Third-Party Sharing

The App does not transfer or share user personal information with third parties, except in the following cases:

5.1 Google AdMob

  • Purpose: Displaying advertisements
  • Shared Items: Advertising identifier (anonymous)
  • Recipient: Google LLC
  • Related Policy: Google AdMob Privacy Policy

6. User Rights

Users can exercise the following rights at any time:

  • Right to Access: View your to-do data within the app
  • Right to Modify: Freely modify to-do data
  • Right to Delete: Delete to-do data or uninstall the app
  • Right to Stop Processing: Stop data processing by uninstalling the app or revoking permissions

7. Permission Usage

7.1 Required Permissions

  • Accessibility Service (BIND_ACCESSIBILITY_SERVICE)
    • Purpose: Providing lock screen alternative functionality
    • Data Collection: None (function use only)

7.2 Optional Permissions

  • Calendar Read (READ_CALENDAR)
    • Purpose: Displaying today's and tomorrow's events
    • If Denied: Calendar feature unavailable (other app features work normally)

8. Children's Privacy

The App is not intended for children under the age of 14 and does not knowingly collect personal information from children.

9. Privacy Officer

Officer: Yoonsik Yoon (Individual Developer)

Contact: ys4yune@gmail.com

10. Changes to Privacy Policy

This Privacy Policy may be changed due to changes in laws, policies, or service improvements. Changes will be announced through in-app notifications or update notes.

11. Data Security

The App takes the following security measures to protect user data:

  • All data is stored only on the device
  • Android sandbox security model applied
  • No data transmission to external servers
  • Regular security updates

12. International Data Transfer

The App does not transfer user personal information internationally. (However, Advertising ID from Google AdMob is processed according to Google's policies)
